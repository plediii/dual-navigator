dual-navigator example

Here are some mock documents without state:

  • document/hello
  • document/hello/world
  • document/foo/bar

Here are some mock documents with state which must be asynchronously closed before moving to the next state.

  • stateful/hello
  • stateful/hello/world
  • stateful/foo/bar

Here are some links which will be redirected. Note that the redirect address will not appear in the browser history.

  • redirect/document/hello
  • redirect/document/hello/world
  • redirect/stateful/hello

The following routes will fail to load

  • This route does not exist: wrong/turn
  • This route will fail to respond: throw/woah/there
  • This route will respond incorrectly: bad/route

See the source code for the routes at https://github.com/plediii/dual-navigator/blob/master/example/routes.js.
